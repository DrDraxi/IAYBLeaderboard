## [1.0.0] - 2026-03-23

### Added
- Friends leaderboard panel on level select and level complete screens
- World leaderboard panel showing #1 globally + players around you
- Grade badges (D/C/B/A/S) matching the game's grading system
- Horde mode support (descending score leaderboards)
- Bulk upload of existing save data on game launch
- Auto-upload on level completion
- Game font (octin college) and UI_MilitarySquare background integration
- Rank badges (#1, #2, etc.) on world leaderboard entries
- Separator between #1 and neighborhood in world panel when not adjacent
- Panels hidden on scene transitions and category select navigation
